What is multiplication

Multiplication means "repeated addition"

Lets take two multiplied by three.
this simply means that we need to add number two three times.
And three multiplied by five, will mean we need to add number three five times

two multiply by three can also be written as 2 * 3
and this can in turn be written as 3 * 2.
This is because 2 * 3 will give us the same answer as 3 * 2.

2 * 3 = 2 + 2 + 2 = 6.
and
3 * 5 = 3 + 3 + 3 + 3 + 3 = 15.
